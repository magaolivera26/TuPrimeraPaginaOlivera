# Vivero Online - Proyecto Final Django

Este proyecto es una aplicación web tipo blog desarrollada con Django como entrega final del curso.

---

## Funcionalidades

- Registro de usuarios
- Login / Logout
- Perfil de usuario con avatar
- Edición de perfil
- Blog con CRUD de posts
- Página About
- Sistema de clientes, productos y pedidos
- Mensajería básica

---

## Perfil de usuario

Cada usuario puede:
- Ver su perfil
- Editar sus datos
- Subir avatar
- Ver su información personal

---

## Blog

- Crear posts
- Editar posts
- Eliminar posts
- Ver listado de publicaciones
- Ver detalle de cada post
- Uso de CKEditor para contenido enriquecido

---

##  About

Página informativa del autor del proyecto accesible desde la navegación.

---
